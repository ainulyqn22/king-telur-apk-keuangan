# HouseERP

HouseERP is a production-oriented monolithic Next.js App Router ERP for a duck-egg business. It tracks farm production, raw and salted egg inventory, production batches, purchases, sales, operating costs, dashboards, analytics, user roles, and reports.

The application is optimized for Vercel SSR and uses Supabase for PostgreSQL, Auth, row-level security, and server-side business commands. Browser code does not access Supabase directly.
